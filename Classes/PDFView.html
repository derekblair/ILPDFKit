PDFView Class Reference

Inherits from UIView
Declared in PDFView.h

Overview

The PDFView class allows for viewing a PDF file. The controller PDFViewController uses PDFView as its view and PDFDocument as its model. PDFView is typically not directly instantiated, but instead is used as the instance that comes with PDFViewController.
